# Sistema de Chamados – SQL

Projeto simples de banco de dados para controle de chamados de suporte técnico.

## Objetivo
Simular um cenário real de suporte técnico, com usuários, chamados e histórico de atendimentos.

## Estrutura
- usuarios
- chamados
- atendimentos

## Funcionalidades
- Controle de chamados em aberto e fechados
- Organização por setor
- Histórico de atendimentos

## Tecnologias
- SQL (MySQL)

Projeto com foco em aprendizado e fundamentos de banco de dados aplicados à área de suporte técnico.
